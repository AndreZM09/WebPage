Process management system

The project is a simulation of a basic process management system found in operating systems, which uses a quantum-based scheduling algorithm that uses Round Robin. The system allows users to define a series of processes that involve random mathematical operations. These processes are then executed in a specific order, and are given "CPU" time based on the quantum value specified by the user. Throughout the execution process, the user can interact with the system and perform various actions such as pausing, continuing, terminating, blocking, or creating new processes.

Data strucures project

We've developed a C++ program for registering people. The first step is to input the number of individuals you want to register. Then, you'll need to fill in all of the user's personal information. The program will then sort each record through lists and produce three text files. The first file will contain every record with all of its attributes, the second file will contain the ID and its primary key of each record, and the third file will contain the ID and the secondary key of each record.

The program can also search a record by either the primary key (RFC) or the secondary key (city). If you search by primary key, it will return only one record because the RFC is unique. However, if you search by city, the program could return several records because more than one person could be living in the same city.
